## Resumo dos comandos
- Cat: lista o conteúdo dos arquivos;
- Tac: Mostra o conteúdo do arquivo em ordem reversa;
- More: Este comando mostra o conteúdo de forma paginada, pressionando a tecla espaço para passar as páginas;
- Less: O comando less pagina o conteúdo e permite buscar palavras.
Ex: cat messages | less
- Sort: Este comando ordena as linhas de um texto;
   * ex: sort -n numeros.txt(ordenar números)
   * ex: sort -n numeros.txt > novonome.txt(*salvar ordenação sorte em arquivo*)
- WC: O comando WC (word call) serve para **contar linhas, palavras e bytes nesta ordem**, de um determinado arquivo;
   * ex: wc -l nomearquivo.txt(**linhas**)
   * ex: wc -w nomearquivo.txt(**palavras**)
   * ex: wc -c nomearquivo.txt(**caracteteres**)
- Cut: Corta seções de um arquivo, exibindo o conteúdo de uma determinada coluna;
  * ex: cut -d(separador): -f1(colunas, útil quando os arquivos tem padrão) cientistas.txt;
  * cut -d: -f2 cientistas.txt;
  * cut -d: -f1,5 /etc/passwd(login(-f1) buscar nome dos usuários(5));
  * cut -d: -f1,7 /etc/passwd(login(-f1) buscar shell dos usuários(7));
- Head: O comando head mostra o inicio do arquivo, ou seja, às 10 primeiras linhas do arquivo;
   * ex: head -n 50 ( mostra as 50 primeiras linhas do arquivo);
- Tail: O comando tail mostra a caúda do arquivo, ou seja, às últimas 10 linhas de um arquivo;
   * ex: tail -n 50 : mostra às 50 últimas linhas de um arquivo.
