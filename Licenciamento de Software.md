# Licencimento de software.
## Modelos de software proprietário.
- Comercial: Obtenção de lucros. Ex: Windows, MS-Office.
- Shareware: Mesma finalidade, mas com algumas funcionalidades podem ser usadas antes de se pagar. #TesteDriver
- Freeware: Geralmente gratuito, com limitações em relação a um modelo pago. Ex: Nginx
## Modelo de software livre.
> Free Software Foundation(FSF);  
> Open Source Iniciative(OSI).  
### Pilares defendidos pela FSF. 
- 1. Liberdade de usar o software para o que quiser;
- 2. Liberdade de ver e alterar o código-fonte;
- 3. Liberdade de redistribuir o software;
- 4. Liberdade de redistribuir sua cópia modificada de software.

### Definição de software livre pela OSI.
- 1. Redistribuição permitida;
- 2. Código-fonte deve ser disponível;
- 3. Deve ser redistribuído no mesmo licenciamento em que foi primeiramente licenciado;
- 4. Nenhuma discriminação de pessoas ou grupos;
- 5. Nenhuma proibição de como o software deve ser usado;
- 6. Não pode depender de outras licenças;
- 7. Nenhuma proibição de rodar em algum produto específico;
- 8. Nenhuma proibição de executar somente em alguma tecnologia.
